  2. Install & run:

  3. Open browser:
     http://localhost:5173

  4. Login:
     Auto-authenticates as demo user "كريم"

  5. Play:
     Click on a course → Select lesson → Complete games

🎮 KEY FEATURES:

  XP System:
    • 5-40 XP per game (varies by difficulty & type)
    • 50-150 XP per lesson (varies by level)
    • Daily challenges: 25-40 XP each
    • Time-Rush multipliers: up to 1.5×

  Hearts System:
    • 5 max hearts
    • 1-2 penalty per wrong answer
    • Passive recovery: 1 heart every 4 hours
    • Flashcard: 0 penalty (self-grading)

  Progression:
    • 60% gate to unlock next level
    • Locked/Unlocked/In Progress/Completed states
    • Error tracking per game
    • Best score saved

  Streaks:
    • Daily activity tracking
    • Resets on missed day
    • Visual indicator (🔥)

  Achievements:
    • First Lesson
    • Week Streak (7+ days)
    • Perfect Game (0 errors)
    • Speed Demon (10× multiplier)
    • Level Master (all lessons in level)
    • Course Master (all lessons in course)

🐛 TROUBLESHOOTING:

  Mock data missing?
    localStorage.removeItem('sadeen.mock.state');
    location.reload();

  Backend not responding?
    1. Check: mongod running?
    2. Check: npm start in backend/
    3. Check: localStorage.getItem('sadeen.mock')

  XP not calculating?
    Check: GamificationService.calculateGameXp()
    Check: GAMIFICATION_CONFIG.xpRewards
